Bunch of git shortcuts for fish shell. [Author][1].
*All commands accept arguments as usual.*

`gi` —  `git init`

`ga` — `git add`

`gs` — `git status`

`gr` — `git rm`

`gc` — `git commit`

`gcl` — `git clone`

`gpll` — `git pull`

`gpsh` — `git push`

[1]:	https://github.com/ubermonkey "Askhat Bikmetov"
